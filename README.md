# Python & Pandas Mini Projects

This repository contains a collection of hands-on mini projects developed while learning **Python** and **Pandas**. The projects focus on applying core programming concepts to real-world scenarios such as **Student Management**, **Employee Management**, **Library Management**, and **Student Result Analysis**. Each project is menu-driven and demonstrates how data can be stored, managed, analyzed, and manipulated using Pandas DataFrames.

These projects were built to strengthen my understanding of data analysis, file handling, user input, and DataFrame operations through practical implementation. They showcase common operations such as searching, sorting, filtering, grouping, adding, updating, deleting records, and exporting data to CSV files.

## Projects Included

* Student Result Management System
* Student Management System
* Employee Management System
* Library Management System
* Other Python and Pandas practice exercises

## Features

* Display records
* Search records by different criteria
* Add, update, and delete records
* Sort records based on multiple fields
* Calculate summary statistics (average, maximum, minimum)
* Group and analyze data using Pandas
* Read from and write to CSV files
* Menu-driven interface using Python `match-case`

## Technologies Used

* Python
* Pandas
* CSV File Handling

## Learning Outcomes

Through these projects, I gained practical experience with:

* Pandas DataFrames
* Data filtering and querying
* Data sorting and grouping
* Aggregate functions (`mean()`, `max()`, `min()`, `sum()`)
* Reading and writing CSV files
* CRUD operations using Pandas
* User input handling
* Menu-driven programming
* Python `match-case` statements

## Purpose

The primary objective of this repository is to document my hands-on learning journey with Python and Pandas by implementing small, real-world management systems. These projects helped me understand how data analysis and manipulation techniques can be applied to practical applications while building a strong foundation in Python programming.
