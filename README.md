Student Result Management System using Python and Pandas

A menu-driven Student Result Management System built using Python and Pandas. 
This project demonstrates basic data analysis and file handling operations such as 
displaying student records, finding the topper, calculating average marks, 
searching students by name, adding new student records, and saving data to CSV files.

Features
Display all student records
Find the topper based on marks
Calculate average marks
Search students by name
Add new student records
Save updated data to a CSV file
Menu-driven interface using Python match-case


Technologies Used
Python
Pandas
CSV File Handling
Learning Outcomes

This project helped in understanding:

Pandas DataFrames
Data filtering and querying
Aggregate functions (mean(), max())
Reading and writing CSV files
